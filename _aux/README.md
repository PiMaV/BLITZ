# Auxiliary content

Benchmarks, example dataset, external posts, scripts, and tests. Kept in one folder to keep the repo root minimal.

- **benchmarks/** – performance and load benchmarks
- **Example_Dataset_kINPen_Surface_100kFPS/** – sample image set for testing
- **external_posts/** – social/marketing snippets (e.g. LinkedIn)
- **scripts/** – standalone utilities (e.g. DICOM converter)
- **tests/** – pytest unit and integration tests
